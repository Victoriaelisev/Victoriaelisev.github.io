PacMan Exercise

In this coding exercise, you will be creating a version of the classic game Pac-Man. The main focus of this exercise will be on animating Pac-Man's movement and mouth opening and closing as he travels across the screen. You will also be implementing collision detection, so that when Pac-Man reaches the end of the screen, he will reverse direction. This project will involve creating a simple game loop and using basic graphics rendering techniques to display Pac-Man on the screen. It will also require you to use conditional statements and basic math to calculate Pac-Man's position and direction of movement.

Eye Exercise

In this coding exercise, you will use JavaScript, HTML, and CSS to create a webpage that features two eyes. The pupils of the eyes will follow the movement of the mouse as it moves around the page. This will be achieved through the use of event listeners in JavaScript to track the mouse's position and update the position of the pupils accordingly. The HTML will be used to structure the webpage and the CSS will be used to style the elements, including the eyes and pupils. The final result will be a webpage with interactive eyes that follow the movement of the mouse.

Real Time Bus Tracking

In this coding exercise, you will create a real-time bus tracking program using JavaScript, HTML, and CSS. The program will display a map of a city, with bus icons representing the location of active buses. As the buses move, the program will update the positions of the icons on the map in real-time. The program will also display information about each bus, such as its route number and destination. The HTML and CSS will be used to structure and style the user interface, while the JavaScript will be used to track the positions of the buses and update the map accordingly. The final result will be a dynamic, interactive program that allows users to track the movements of buses in a city in real-time.
